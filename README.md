## 🧠 TIL 작성 방식
- 매일 저녁 19:00 ~ 21:00 사이 작성
- 기본 포맷: `문제점 / 시도한 것 / 해본 것 / 알게 된 것` 구조
- 혼잣말, 자문자답 스타일로 작성
- 마크다운 기반 정리, 블로그와 병행 업로드
